# App+ #

App+是一个主要用于Android用户间传送Apk文件的工具软件。

### 截图 ###

![index](/art/index.png "")
![send](/art/send.jpg "")<br>
![setting](/art/setting.png "")
![theme](/art/theme.png "")

### 知识点 ###
* 主题切换
* Logger管理
* 友盟统计 包括事件统计以及自动更新
